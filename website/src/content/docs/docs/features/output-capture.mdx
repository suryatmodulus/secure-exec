`exec()` runs guest code and returns its captured output. The `stdout` and `stderr` streams are buffered separately into strings, and the guest process's `exitCode` is returned alongside them. Buffering into strings bounds host memory: `exec()` holds each channel as a single string rather than retaining an unbounded backlog of live stream chunks.

## Capturing Output

```ts
import { NodeRuntime } from "secure-exec";

const rt = await NodeRuntime.create();

try {
	const { stdout, stderr, exitCode } = await rt.exec(`
		console.log("hello from the sandbox");
		console.error("oops from the sandbox");
		process.exit(3);
	`);

	console.log("exitCode:", exitCode);
	console.log("stdout:", JSON.stringify(stdout));
	console.log("stderr:", JSON.stringify(stderr));
} finally {
	await rt.dispose();
}
```

*[See Full Example](https://github.com/rivet-dev/secure-exec/tree/main/examples/docs/feat-output-capture)*

Running it prints:

```
exitCode: 3
stdout: "hello from the sandbox\n"
stderr: "oops from the sandbox\n"
```

`stdout` and `stderr` are independent strings, so output written to one channel never bleeds into the other. The `exitCode` is the guest process's exit status. Here the program called `process.exit(3)`, so `exitCode` is `3`.

## Result shape

`exec()` resolves to an object with three fields:

```ts
interface NodeRuntimeExecResult {
	stdout: string;
	stderr: string;
	exitCode: number;
}
```

:::note[Streaming output is not on `NodeRuntime` yet]
`NodeRuntime.exec()` returns the full `stdout` and `stderr` strings once the
guest process finishes. Secure Exec supports incremental streaming underneath
(the kernel's exec protocol carries live stdout/stderr stream events), but
`NodeRuntime` does not surface per-chunk callbacks yet, so the buffered strings
are what you get today.
:::

## Common patterns

Use `exitCode` to detect failure, and `run()` when you also need a value back (it returns the same `stdout`/`stderr`/`exitCode` plus a decoded `value`):

```ts
const { stderr, exitCode } = await rt.exec(code);
if (exitCode !== 0) throw new Error(`guest exited ${exitCode}: ${stderr}`);

const { value } = await rt.run(`globalThis.__return({ answer: 42 });`);
// value === { answer: 42 }
```
